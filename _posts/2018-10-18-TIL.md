1. 사람이 C 언어로 소스코드를 짜면, 컴파일러를 통해 컴퓨터가 알아 들을 수 있는 1과 0으로 번역해줘야한다. (소스 코드 - 컴파일러 - 머신 코드)

2. body에서 CSS를 적용할 부분을 선택할 때(선택자) 태그, 아이디, 클래스 중에서 선택한다. 태그의 경우 태그 명칭을 바로 적으면 되고, 아이디는 #아이디, 클래스는 .클래스로 적는다. 

  ```html
  <style>
      p{ color: red;}
      #id_name{ font-weight: bold;}
      .class_name{ font-size: 20px}
  </style>
  ```
3. span 태그는 웹브라우저로 볼 때는 아무 차이 없지만, span 태그로 묶으면 그 부분에 스타일을 적용하기 좋다. 

   ```html
   <style>
       span{ color: green;}
   </style>
   ```
   ```html
   <body>
       <p>
           강조하고 싶은 <span>부분</span>만을 span으로 묶는 다던지...
       </p>
   </body>
   ```
