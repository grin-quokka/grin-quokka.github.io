** 오늘 배운 것 **

* IP Address : 집 주소 처럼 인터넷에서 통하는 고유 주소 (IPv4: 4 cluster of 8 bits -> IPv6: 8 cluster of 16 bytes)
* DHCP (Dynamic HOst Configuration Protocol) server : IP주소 할당해주는 역할
* DNS (Domain Name System) server : IP 주소를 직접 입력해서 찾아가기 번거로우니까 도메인 이름을 쓴다. 
* Access Points : 모든 기계가 다 고유한 IP 주소를 갖는게 아니라, 하나의 라우터를 이용하는 기계들은 다 같은 IP 주소를 갖도록 한다.
